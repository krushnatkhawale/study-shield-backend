# TV Device Service - Detailed Description

## Purpose
Manage Android TV connections and network history.

## Key Responsibilities
- Store connected TVs per WiFi network
- Maintain history of networks and TVs
- Provide current network + available TVs to mobile app

## Current App Adaptation Note
Enhance the existing TV connection logic. Add history without breaking current pairing functionality.

- WifiNetwork
- ConnectedTV
